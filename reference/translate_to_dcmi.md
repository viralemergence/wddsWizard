# translate to dcmi

translate to dcmi

## Usage

``` r
translate_to_dcmi(item, translation_map)
```

## Arguments

- item:

  List. Item to be translated.

- translation_map:

  List. Instructions for translating the item

## Value

List. Item that has been translated to DCMI
