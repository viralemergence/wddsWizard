# Becker et al. project metadata

The project metadata for a bat coronavirus dataset that conforms to the
wildlife disease data standard. See data standard for field
descriptions.

## Usage

``` r
becker_project_metadata
```

## Format

An object of class `list` of length 11.

## Source

<https://www.ebi.ac.uk/pride/archive/projects/PXD031075>
