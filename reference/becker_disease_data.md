# Becker et al. dataset

A bat coronavirus dataset that conforms to the wildlife disease data
standard. See data standard for field descriptions

## Usage

``` r
becker_disease_data
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 2
rows and 21 columns.
