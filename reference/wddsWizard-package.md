# wddsWizard: Data Wizard for a Minimal Wildlife Disease Data Standard

Facilitates compliance with and use of the Wildlife Disease Data
Standard stored on Zenodo <https://doi.org/10.5281/zenodo.15020049>. It
allows users to restructure and validate datasets.
