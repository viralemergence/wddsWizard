# WDDS to PHAROS metadata mapping

A table that maps variables between WDDS and the PHAROS data standard
(11 September 2025). Will be deprecated once PHARSO and wdds are
aligned.

## Usage

``` r
wdds_to_pharos_map
```

## Format

An object of class `spec_tbl_df` (inherits from `tbl_df`, `tbl`,
`data.frame`) with 45 rows and 2 columns.

## Source

<https://pharos.viralemergence.org/>
